# Ecash escrow

## Structure

```
.
├── ...
├── apps
│   ├── telegram-ecash-escrow                  # NextJS App
├── packages
│   ├── api                   # Shared API config & models
└── ...
```

## Development

You can run all apps at once:

```bash
pnpm dev
```

... or run one app only:

```bash
pnpm dev:next
```

Make sure to refer to the available scripts in `package.json`.
